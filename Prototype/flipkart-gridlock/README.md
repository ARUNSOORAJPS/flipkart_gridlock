## Development

The SiMPL 2 Mobile stack is based on the [Ionic CLI](http://ionicframework.com/docs/v2/cli/)
which is automatically installed when following the getting started guide. Use
the [*official Ionic documentation*](http://ionicframework.com/docs/v2) to get
more information.

### Setup project

To work with this application, clone the sources from GIT and install the
nessesary dependencies using NPM as following:

``` sh
git clone 
cd hono-demo
npm install
```

You can now use all commands provided by the Ionic CLI and start developing.

### Start application

Run the app and play around in the browser:

``` sh
ionic serve
```

You can use `ionic serve --lab` to compare android and ios styled versions in
one screen.
